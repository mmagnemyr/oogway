Oogway Components

Oogway consists of
  • Code Template Files
    • There are code templates for the VGCS ODL Team AWS Steps
    • You can provide new code templates for other platforms.
  • Documentation Template Files, Located in \oogway\documentaion_templates\{orchestration language}
    • html templates
    • the templates are used by oogway when building the documentaion for your use cases.
  • Excel Templates Files. Located in \oogway\templates\
    The vXXX in the file names mentioned below is refering to the version number the files has. If the current version is 117 then solution_vXXX.xlsx is actually solution_v117.xlsx
    • oogway_sql_vXXX.xlsx
      • This is where most of the oogway-variables are calculated.
    • oogway_table_template_vXXX.xlsx
      • This is where use case developers defines their table
    • solution_vXXX.xlsx
      • This is common meta data for the use cases that are shared for all tables.
  • Oogway Python Script. Located \oogway\code_generator
      oogway.py
